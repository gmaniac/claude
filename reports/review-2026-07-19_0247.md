- Evidence: installed 9.0.17 (`installed_plugins.json`, installed 2026-02-06); upstream `thedotmack/claude-mem` latest release is **v13.11.0** (GitHub API); the thedotmack marketplace manifest hasn't refreshed since 2026-02-06, so "matches marketplace latest" is vacuously true. This was the plugin flagged "watch" for SessionStart bloat (currently 16.8KB, down from 17.9KB last week).

### skills: `open-gstack-browser` wrapper duplicates `connect-chrome`

### plugins: beads plugin skills reference `bd dolt`, which the installed bd CLI doesn't have
